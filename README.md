# Implement Gemini API for generate things using Express js

### Endpoints

_Below is a list endpoints for this project._

1. Generate Text
   ```
   /generate-text
   ```
2. Generate Image
   ```
   /generate-from-image
   ```
3. Generate Document
   ```
   /generate-from-document
   ```
4. Generate Audio
   ```
   /generate-from-audio
   ```
